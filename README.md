CampusX

A Smart Campus Management System designed to simplify student life by integrating notes, events, complaints, notifications, lost & found, and AI-based features into one platform.

🚀 Features
📚 Notes Sharing System
📅 Event Management
⚠️ Complaint System
🔔 Notification Panel
🔍 Lost & Found System
🤖 AI Assistant (Upcoming Feature)
👤 User Authentication (Login / Signup UI)
📊 Dashboard Overview
🛠️ Tech Stack
HTML5
CSS3
JavaScript
Font Awesome Icons
Responsive Web Design
🎯 Purpose

CampusX is built to:

Digitize campus operations
Improve student communication
Centralize academic resources
Provide a modern smart-campus experience
